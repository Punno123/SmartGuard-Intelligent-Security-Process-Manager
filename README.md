# SmartGuard-Intelligent-Security-Process-Manager

SmartGuard is a Python-based process monitoring application that helps detect suspicious system processes based on CPU usage, thread count, process status, and other risk indicators. It provides a user-friendly Tkinter GUI to view active processes, search by process name, log suspicious activity, and safely terminate selected processes with confirmation. The project uses `psutil` for real-time process monitoring and includes basic protection against terminating critical system processes.
